# Autho Frontend

## Что поддерживает

- `POST /api/v1/signup` — регистрация 
- `POST /api/v1/login` — логин 
- `GET  /api/v1/` — получить профиль текущего пользователя
- `POST /api/v1/logout` — выйти

## Быстрый старт (dev)

1) Установить зависимости:

```bash
npm i
```

2) Запуск:

```bash
npm run dev
```

Открыть: `http://localhost:5173`
